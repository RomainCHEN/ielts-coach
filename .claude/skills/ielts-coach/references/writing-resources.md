# IELTS Writing Resources

## Task 1 (Academic) - Chart Description

### Common Chart Types
| Type | Key Language |
|------|-------------|
| Line graph | trends, fluctuations, peaked at, plateaued, rose steadily, declined sharply |
| Bar chart | compared to, the highest, the lowest, accounted for, significantly more/less |
| Pie chart | proportion, percentage, the majority of, a minority of, comprised, consisted of |
| Table | figures, data indicates, according to the table, is illustrated by |
| Process diagram | initially, subsequently, the next stage involves, finally, this results in |
| Map | was replaced by, was converted into, underwent redevelopment, remained unchanged |

### Essential Task 1 Structure
1. **Introduction** (1 sentence): Paraphrase the question
2. **Overview** (2-3 sentences): Main trends / key features (NO data)
3. **Detail Paragraph 1** (3-4 sentences): Specific data points, comparisons
4. **Detail Paragraph 2** (3-4 sentences): Remaining features

### Must-Use Language for Task 1
```
Rise:      increased, rose, grew, climbed, soared, surged
Fall:      decreased, declined, dropped, fell, plummeted, dipped
Stability: remained stable, levelled off, stayed constant, plateaued
Fluctuation: fluctuated, varied, oscillated
Peak:      peaked at, reached a peak of, hit a high of
Low:       bottomed out at, reached a low of

Adverbs:   sharply, dramatically, significantly, steadily, gradually, slightly
Adjectives: sharp, dramatic, significant, steady, gradual, slight

Comparison: while, whereas, in contrast, compared to, on the other hand
Similarity: similarly, likewise, both...and..., in the same way
```

### Band-Specific Task 1 Requirements
**Band 6**: Clear overview, key features presented, some data support, adequate vocabulary
**Band 7**: Clear overview, key features clearly presented, logical organization, good range of vocabulary
**Band 8**: All requirements of the task covered, clear overview, well-selected data, wide vocabulary range

## Task 2 - Essay Structure

### Essay Types
1. **Opinion** (To what extent do you agree or disagree?)
2. **Discussion** (Discuss both views and give your opinion)
3. **Problem-Solution** (What are the problems and solutions?)
4. **Advantages-Disadvantages** (Do the advantages outweigh the disadvantages?)
5. **Two-part Question** (Why is this? What can be done?)

### Universal Essay Template

```
Introduction (40-50 words)
├── Hook: General statement about the topic
├── Context: Why this is important/relevant
└── Thesis: Your position + outline of body paragraphs

Body Paragraph 1 (80-100 words)
├── Topic Sentence: Main idea
├── Explanation: Develop the idea
├── Example: Specific, concrete example
└── Link: Connect back to thesis / transition

Body Paragraph 2 (80-100 words)
├── Topic Sentence: Main idea
├── Explanation: Develop the idea
├── Example: Specific, concrete example
└── Link: Connect back to thesis

Conclusion (30-40 words)
├── Restate position (use different words)
├── Summarize key points
└── Final thought / recommendation
```

### Band-Specific Essay Requirements
**Band 6**: All parts addressed, clear position, adequate paragraphing, some complex sentences
**Band 7**: All parts addressed, clear position throughout, logical organization, range of complex structures
**Band 8**: Well-developed response, extended ideas, skillful paragraphing, wide range of structures

### Topic-Specific Vocabulary Banks
Use the essay topic to pull relevant vocabulary. Always include 5-8 topic-specific terms per essay.
