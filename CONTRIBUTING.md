# Contributing to IELTS Coach

Thanks for helping make IELTS preparation less stressful!

## Bug Reports

Found a topic inaccuracy, missing question, or broken behavior? Open an issue with:

- Which topic/question is wrong or missing
- What you expected vs what happened
- Your IELTS season (e.g., May-August 2026)

## Feature Suggestions

Have an idea? Open an issue with the `enhancement` label. Popular areas for contribution:

- HTML design improvements
- Additional sample answers for the calibration reference pool
- New question bank seasons
- Writing Task 1 chart type vocabulary expansions

## Pull Requests

1. **Fork** the repo and create a branch from `main`
2. **Follow commit conventions:** Use `feat:`, `fix:`, `docs:`, `refactor:`, `chore:` prefixes
3. **Keep changes scoped** — one PR = one change
4. **Update both READMEs** if your change affects documented behavior
5. **Test with the skill** — run a real session before opening the PR

## Skill Design Principles

When contributing to `SKILL.md`, keep these in mind:

- **Agent does the work, user provides the content.** Never offload configuration to the user.
- **Polish, don't replace.** All generated answers must preserve the user's voice.
- **Band-calibrate, don't inflate.** Match the target score, not impress with complexity.
- **Anti-AI-flavor always.** Any new answer format must pass the human-writer test.
